Google Analytics Setup

To enable traffic analytics, replace the placeholder GA4 ID in index.html with your actual Measurement ID.

Steps:
1. Go to analytics.google.com
2. Create a new property → Web stream
3. Copy the Measurement ID (format: G-XXXXXXXXXX)
4. Replace G-XXXXXXXXXX in two places in index.html
